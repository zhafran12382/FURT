## 📖 Syntax Highlights

### Natural Language Variables
```furt
name becomes "Alice"
score starts at 100
health stays at 80
```

### Natural Language Conditions  
```furt
when score is at least 100:
    show "High score!"

when health is less than 20:
    show "Low health warning!"
```

### Natural Language Loops
```furt
repeat 5 times:
    show "Hello!"

count from 1 to 10:
    show "Number:", counter
```

### Graphics Commands
```furt
use graphics
make_window 800 600
set background to light blue
draw_circle 400 300 50
```
